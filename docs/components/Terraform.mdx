## Instructions

## Terraform Configuration

To use the Terraform integration, you need an HCP Terraform API Token.

1. Go to the [Team API Tokens page](https://app.terraform.io/app/&lt;YOUR_ORG&gt;/settings/authentication-tokens?tabIndex=1) in HCP Terraform, replacing "&lt;YOUR_ORG&gt;" with your actual organization name.
2. Click **Create a team token** and provide a description.
3. Once generated, paste the token in the API Token field below.

## Run plan

Creates a Terraform run, polls its status, and waits for the plan to complete. It creates a plan only and does not allow applying.

### Example Output

```json
{
  "finalStatus": "planned",
  "runId": "run-xxxxx"
}
```
