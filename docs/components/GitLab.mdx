## Instructions

For **App OAuth**, leave **Client ID** and **Secret** empty to start the setup wizard.

For **Personal Access Token**, use scopes: api, read_user, read_api, write_repository, read_repository.

## Create Issue

The Create Issue component creates a new issue in a specified GitLab project.
