## On Issue

The On Issue trigger fires when issue events occur in a GitLab project.

### Configuration

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| **Project** | Resource | Yes | The GitLab project to monitor |
| **Actions** | Multi-select | Yes | Issue actions to trigger on: `open`, `close`, `reopen`, `update` |

### Example Payload

```json
{
  "object_kind": "issue",
  "event_type": "issue",
  "object_attributes": {
    "id": 301,
    "iid": 1,
    "title": "Example Issue",
    "state": "opened",
    "action": "open",
    "url": "https://gitlab.com/group/project/-/issues/1"
  },
  "user": {
    "id": 1,
    "name": "John Doe",
    "username": "johndoe"
  },
  "project": {
    "id": 15,
    "name": "my-project",
    "path_with_namespace": "group/my-project"
  }
}
```
